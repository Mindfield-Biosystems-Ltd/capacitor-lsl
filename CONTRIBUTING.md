# Contributing to @mindfield/capacitor-lsl

Thank you for your interest in contributing! This plugin is maintained by Mindfield Biosystems Ltd.

## Development Setup

1. Clone the repository
2. Install dependencies: `npm install`
3. Build the TypeScript: `npm run build`

## Building

```bash
# TypeScript + Rollup bundle
npm run build

# Watch mode (TypeScript only)
npm run watch
```

## Code Style

- TypeScript: strict mode, ES2017 target
- Kotlin: standard Android conventions, JVM 17
- Objective-C: standard iOS conventions, iOS 14.0+

## Pull Requests

1. Fork the repository
2. Create a feature branch from `main`
3. Make your changes
4. Run `npm run build` to verify TypeScript compiles
5. Submit a pull request with a clear description

## Reporting Issues

Please use [GitHub Issues](https://github.com/Mindfield-Biosystems-Ltd/capacitor-lsl/issues) and include:
- Plugin version
- Capacitor version
- Platform (Android/iOS) and OS version
- Steps to reproduce
- Expected vs actual behavior
